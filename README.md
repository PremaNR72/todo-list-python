# Todo List Application

## GitHub Repository
This project is uploaded to GitHub for learning and portfolio purposes.

## Project Description
This is a simple Todo List application built using Python and Tkinter.
It allows users to add, delete, and manage daily tasks.

## Features
- Add tasks
- Delete tasks
- Save tasks in file
- Load tasks when the program starts

## Technologies Used
- Python
- Tkinter
- File Handling

## How to Run the Project
1. Install Python
2. Download the project
3. Run the command:

python main.py
